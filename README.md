# EasyRecyclerView
EasyRecyclerView
Getting Started

To use this library, add the following to your project level build.gradle:
 ```
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
    ```
Add this to your app build.gradle:
**EasyRecyclerView**
 	--------
    ```
    dependencies {
       compile 'com.github.cgpllx:EasyRecyclerView:v1.0.0'
    }
    ```
